# Web prototype

Откройте [index.html](/home/wlad/Projects/Python/ttf2rastr/Web/index.html) в браузере, выберите локальный `.ttf`/`.otf`, задайте параметры и нажмите `Сгенерировать`.

Особенности текущей версии:

- работает полностью локально, без Python и без сервера
- рендерит шрифт в браузере через `FontFace` + `canvas`
- генерирует header в формате, совместимом с основным проектом
- preview рисуется уже из упакованных bitmap-данных, а не через системный текстовый рендер
- если нужен пробел в итоговом шрифте, его надо явно включить в поле `Символы`, иначе он будет считаться отсутствующим глифом

Ограничения прототипа:

- метрики могут не совпасть побитово с `Pillow`, потому что тут другой рендеринг
- качество зависит от поддержки `FontFace` и `actualBoundingBox*` в браузере
- это отдельная веб-ветка функциональности, а не замена `main.py`
